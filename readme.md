# Online Pixel Dungeon

A real-time multiplayer roguelike dungeon crawler based on **Shattered Pixel Dungeon** by Evan Debenham.

50 floors, bosses every 5, full real-time multiplayer — no turns, no waiting.

**🎮 Play now:** [web-352716452401.europe-west1.run.app](https://web-352716452401.europe-west1.run.app)

## Stack

- **Frontend:** React + Vite, canvas rendering
- **Backend:** FastAPI, WebSockets, 20 Hz game loop

## Running locally

```bash
docker compose up
```

- Frontend: http://localhost:3000
- Backend: http://localhost:8080
